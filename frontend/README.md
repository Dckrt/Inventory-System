# 🧋 Milk Tea Inventory System

A full-stack web-based inventory system designed to help milk tea shops manage their stock efficiently.

---

## 🚀 Features

- 🔐 User Login (Admin & Staff)
- 📦 Inventory Management
- ➕ Stock In / ➖ Stock Out
- 📊 Dashboard Overview
- 📈 Reports (Charts)
- 🔍 Search & Filter
- 👤 Role-Based Access Control
- 🌙 Dark Mode UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MySQL (XAMPP)
